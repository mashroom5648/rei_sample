// クローン作成　方法（ターミナルを使用）
1. ターミナルで"git clone ssh://github.com/mashroom5648/rei_sample.git"を入力し、Enterを押す
2. ホームディレクトリ直下にクローンが作成される

// クローン作成　方法（SouceTree）※推奨
1. 開いた最初のウィンドウで「新規」から「URLからクローン」を選択
2. 「ソースURL」に"ssh://github.com/mashroom5648/rei_sample.git"を入力
3. 「保存先のパス」でクローンの保存先を指定（最後の/["ディレクトリ名"]の名前のフォルダが勝手に作成されるので、既存のフォルダ名は使用しない）
4. クローンを押してクローン完了
